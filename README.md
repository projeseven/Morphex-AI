# 🤖 Morphex-AI

**Türkçe yapay zeka sohbet ve kod asistanı**

Morphex-AI, geliştiriciler için tasarlanmış, Türkçe dilinde kod yazma, inceleme, hata bulma ve sohbet yeteneklerine sahip bir web tabanlı yapay zeka asistanıdır.

## 🎯 Amaç

Türkçe konuşan yazılımcılara:
- 💻 Kod yazma ve inceleme
- 🐛 Hata bulma ve düzeltme
- 💬 Sohbet yapma
- 🎤 Ses desteği (Türkçe erkek sesi)

## ✨ Temel Özellikler

- ✅ **Kod Asistanı**: Python kodlama, inceleme, hata analizi
- ✅ **Sohbet Yeteneği**: Rastgele sorulara Türkçe cevap verme
- ✅ **Ses Desteği**: Türkçe metin-ses dönüşümü (erkek ses)
- ✅ **Modern Arayüz**: Güzel giriş/kayıt sayfaları
- ✅ **Web Tabanlı**: Herhangi bir cihazdan erişim

## 🛠️ Teknoloji Stack

- **Backend**: Python
- **Frontend**: Web Teknolojileri (HTML/CSS/JavaScript)
- **Ses**: Türkçe TTS (Erkek Sesi)
- **Platform**: Web Sitesi
- **Dil**: Türkçe
