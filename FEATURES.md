# Morphex-AI - Özellikler ve Gereksinimler

## Proje Tanımı
Morphex-AI, bir kod yazma, kod inceleme, hata bulma, hata düzeltme ve sohbet yapay zekasıdır.

## Temel Özellikler

### 1. **Kod İşlemleri**
- ✅ Kod yazma
- ✅ Kod inceleme
- ✅ Hata bulma
- ✅ Hata düzeltme
- ✅ Python dilinde herhangi bir şeyi kodlayabilir (kötü amaçlı yazılım dışında)

### 2. **Sohbet Özelliği**
- ✅ Rastgele sorulara cevap verebilen yapay zeka
- ✅ Konuşma yeteneği

### 3. **Arayüz Özellikleri**
- ✅ Güzel giriş/kayıt arayüzü
- ✅ Ses desteği (Türkçe erkek sesi)

### 4. **Dil ve Çevre**
- ✅ **Türkçe dil desteği**
- ✅ **Erkek ses (Türkçe TTS)**
- ✅ **Web sitesi olarak geliştirilecek** (masaüstü uygulaması değil)
- ✅ **Backend: Python** (daha iyiyse diğer diller de kullanılabilir)

## Teknik Stack

| Bileşen | Teknoloji |
|---------|-----------|
| Backend | Python |
| Frontend | Web Teknolojileri (HTML/CSS/JavaScript) |
| Ses | Türkçe TTS (Text-to-Speech) - Erkek Sesi |
| Dil | Türkçe |
| Platform | Web Sitesi |

## Yapılacak İşlevler

1. **Kullanıcı Yönetimi**
   - Giriş (Login)
   - Kayıt (Register)
   - Oturum yönetimi

2. **Kod Asistanı**
   - Kod yazma
   - Kod inceleme
   - Hata analizi
   - Hata düzeltme önerileri

3. **Sohbet Fonksiyonu**
   - Genel sorulara cevap verme
   - Konuşma geçmişi

4. **Ses Desteği**
   - Türkçe metin-ses dönüşümü (TTS)
   - Erkek ses

5. **Veri Depolama**
   - Konuşma kaydı
   - Kullanıcı verileri
   - Kod örnekleri
